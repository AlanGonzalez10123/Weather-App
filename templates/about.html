About This Project

Used python and flask to fetch weather data from openweather API. Displayed data and received user input using HTML and applied basic styling with CSS. Used JavaScript and Geolocation API to fetch user location and then use it for weather retrieval. Straighforward to use page. Simply type in the city you want the current weather from and click 'generate'. ' You will then be redirected to a page displaying some current weather data relating to the city the user input. If the user clicks 'Use My Location', the input will autofill and redirect automatically to search for the weather of the User's current location.
